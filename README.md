# Taskify App v.3.1

The app allows to:
- **create**, **edit** and **delete** tasks;
- **mark** tasks as completed;
- **change** completed tasks status to inclomplete;
- **search** through tasks by their titles
- **filter** tasks to show only complete/only incomplete/all
